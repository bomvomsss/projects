# MUZIT — 프로젝트 개요

창작자를 위한 작품 발행 및 감상 플랫폼.
소설·만화 등 다양한 창작물을 올리고, 독자가 읽고 즐길 수 있는 공간.

---

## 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | React 19 + Vite |
| 라우팅 | React Router DOM v7 |
| 에디터 | Tiptap v3 (ProseMirror 기반 리치 텍스트 에디터) |
| 스타일 | 순수 CSS (App.css, WritePage.css) |
| 패키지 매니저 | npm |

---

## 페이지 및 기능

### 1. 홈 `/` — 전체 작품 피드
- 등록된 작품을 **갤러리 그리드(카드형)**으로 표시
- 카드 클릭 시 상세 페이지로 이동
- 카드 구성: 표지 이미지, 카테고리 뱃지, 제목, 작성자

### 2. 작품 작성 `/write` — 블로그 에디터
Tiptap 기반의 리치 텍스트 에디터.

**툴바 기능**
- 텍스트 서식: 굵게, 기울임, 밑줄, 취소선, 인라인 코드
- 제목: H1 / H2 / H3
- 목록: 순서 없는 목록, 순서 있는 목록
- 블록: 인용구, 코드 블록
- 정렬: 왼쪽 / 가운데 / 오른쪽
- 삽입: 링크 URL, 이미지 URL
- 기타: 구분선, 실행취소 / 다시실행

**기타 기능**
- 글자 수 · 단어 수 실시간 카운터
- 표지 이미지 URL 입력 (선택)
- 태그 입력 (쉼표 구분)
- 전체 공개 / 댓글 허용 옵션 체크박스
- 발행 시 HTML 형태로 저장

### 3. 작품 상세 `/post/:id` — 작품 열람
- 제목, 부제목, 작성자, 날짜 표시
- 에디터에서 작성된 HTML 콘텐츠를 그대로 렌더링
- 태그 목록 표시
- "← 목록으로" 뒤로 이동 버튼

### 4. 컬렉션 `/collection` — 즐겨찾기 (예정)
- 좋아요하거나 보관한 작품 모아보기 (미구현)

### 5. 내 서재 `/library` — 내 작품 관리
- `author === "my_account"` 기준으로 내가 쓴 글만 필터링해 표시
- 작품이 없을 경우 안내 메시지 출력

### 6. 알림창 `/notifications` — 알림 (예정)
- 댓글, 공지사항 알림 (미구현)

## 데이터 구조

현재는 React `useState`로 인메모리 관리 (새로고침 시 초기화).

```js
// 작품(Post) 객체
{
  id: number,          // Date.now() 또는 수동 지정
  title: string,       // 작품 제목
  author: string,      // 작성자 (발행 시 "my_account" 고정)
  content: string,     // 에디터에서 생성된 HTML
  coverUrl: string,    // 표지 이미지 URL
  tags: string,        // 쉼표 구분 태그 문자열 (예: "판타지, 소설")
  isPublic: boolean,   // 전체 공개 여부
  allowComments: boolean, // 댓글 허용 여부
}
```
